K-Digi Receipt Bot
Automated Slip Verification & Receipt Delivery via LINE

Customers paying through K-Digi App couldn't find their receipts — the Support team had to manually search and send every one. I built a full end-to-end system: send a slip photo to LINE → OCR reads it → invoice retrieved from KLN API → receipt PDF delivered back in under 5 seconds. Fully automated, with a complete admin dashboard for edge cases.

PHASE 01
n8n Prototype
Built a 26-node n8n workflow to validate the concept and test LINE → OCR → KLN API integration before writing a single line of Python.
PHASE 02
Python + FastAPI
Migrated to production Python 3.12 + FastAPI for full control: custom error handling, role-based dashboard, and proper database logging.
Typhoon OCR Pipeline — Base64 image → extract Ref No. (12-digit) + Amount via 4-pattern regex
3-tier Error Handling — OCR fail, ref_error, invoice_error each routed automatically
KLN Internal API — secure HTTP POST for invoice URLs, no direct file server access
Role-based Dashboard — SHA-256+salt auth, 8hr sessions, Double-claim Guard
LINE Rich Menu — 3-button menu with Flex Message invoice delivery
Shared Quick Reply — team-wide canned responses stored in SQLite DB
